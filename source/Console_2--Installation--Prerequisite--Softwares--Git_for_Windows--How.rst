How to do it
============

Installation
------------

Download
````````

From the project page
^^^^^^^^^^^^^^^^^^^^^

    * `Download Git for Windows`_.

Install
```````

    * Launch the installer and follow the instructions.

Post Installation
-----------------

Add Git for Windows to the environment variables
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

    * You have to add Git for Windows to your `environment variables`_.
    * This is the path you have to add
    
    ::
    
        <YourDrive>:\Path\to\GitforWindows\bin\
