Color Picker
============

Why do you have to use it
-------------------------

If you are a graphic designer you have, everyday, to deal with colors and need a tool to efficiently pick colors, generate and manage color collections. 

Better if this the tool displays color definition in **HSV**, **RGB**, **CMYK** and **Hexadecimal**.
